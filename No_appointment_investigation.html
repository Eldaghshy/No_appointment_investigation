Introduction¶

-The file of the dataset contains information about 100k medical appointments in Brazil to answer a question whether patients come to the appointment or not and what are their characteristics.Why?

=to make a proper statistical analysis for the reasons that might be the cause of patients not showing up.

-Can it be that the place where the patients live can be the factor ?

-Can it be because of having any handicaps or chronic diseases?

-Can alcohol be the reason for this?

-Or maybe the SMS they receive don't encourage the patients to come to the appointment?

_The age might be a factor perhaps?

Through this project you will be to see if any of these characteristics can a factor or not either by negative correlation or positive correlation. So let's find this out.

#corry is a function that looks for correlation between the variable and each other where:
    #para is the parameter we will use to find correlation with showing up to the appointment or not showing up.
    #data_1 will refer to the patients 'showing up' , data_2 will refer for the patients 'not showing up'.
    #cory_1 and cory_2 represents the plot coming out of data we inputted to be inspected letting us see different aspects 
    #of the data:
 
        #cory_1 will be made in blue with alpha (which reflects the transparency) into 0.4 and and putting the 'Showing up' 
        #as label which will be used for the legend as well as they will represent patients showing up.
            
        #cory_2 will be made in red with alpha set to 0.7 and and putting the 'Not showing up' 
        #as label which will be used for the legend as well as they will represent patients not showing up.

    #Also we have legends by using plt.legend() that depends on the labels I stated in cory_1 and cory_2 for the colors 
    #to give insight about the plot information.

#Both cory_1,_2 will be combined for 2D plot for more exploratory data analyis as the output of this function by using return cory_1,cory_2.      
#Using those colors,labels, alpha and legend they visualization will be clear and easy to be presented to the viewer.
def corry(data_1,data_2,para):
    cory_1=data_1[para].value_counts().plot(kind='bar',alpha=0.4,color='blue',label='Showing up',title='Patients showing up according to {}.'.format(para))
    cory_2=data_2[para].value_counts().plot(kind='bar',alpha=0.7,color='red',label='Not showing up')
    plt.legend()
    return cory_1,cory_2
